# Medicine Reminder Web App

## Project Overview  
A simple and intuitive web application built with the MERN stack (MongoDB, Express.js, React, Node.js) designed to help users manage their medication schedules effectively by setting timely reminders.

## Key Features  
- Add, edit, and delete medicine reminders with specific time and dosage details.  
- User-friendly React frontend for easy interaction and real-time updates.  
- Express and Node.js backend handling API requests and MongoDB database for secure data storage.  
- Notification alerts to remind users when to take their medicines on schedule.

## Technologies Used  
- **Frontend:** React, Tailwind CSS (optional)  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB with Mongoose ORM  
- **Deployment:** (Add your deployment platform if applicable, e.g., Vercel, Heroku)

## Installation & Usage  
1. Clone the repository.  
2. Run `npm install` in both client and server folders to install dependencies.  
3. Set up your MongoDB connection string in a `.env` file.  
4. Run `npm start` in server and client directories to start backend and frontend.  
5. Access the web app on `http://localhost:3000`.
